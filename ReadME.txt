Made by Juho Mänttäri

The course project is located in the project folder
Project can be run after running "npm install" inside mern folder and frontend folder. 
After this the project can be run using "npm run dev" inside mern folder, and the project can be opened using the localhost link in the terminal
MongoDB and Node.js are also required to be installed before

The project itself is an addition/change to mern-stack tutorial exercise.
New additions/changes:
Goals are now "posts", and are ordered from newest to oldest
Users are still only ones that can add/delete their own posts, but they are now viewable to everybody
Add a user list page that shows every user, and clicking on their name links to their posts / user page
